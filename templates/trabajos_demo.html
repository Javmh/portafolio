PROYECTO 1: Analisis de Datos de Ventas

Este análisis ofrece una visión integral del desempeño de ventas, basado en datos específicos de transacciones recientes, y se centra en la evaluación de clientes, productos, mercados geográficos y tendencias temporales. Los puntos clave incluyen las ventas totales, unidades vendidas, y el comportamiento de compra por cliente, producto y país.

Video: Dashboard de Visualización de Datos

Herramientas Utilizadas:

Este análisis y las visualizaciones fueron creados utilizando Power BI y SQL Server, lo que permite integrar datos en tiempo real y realizar análisis detallados para facilitar decisiones estratégicas.

PROYECTO 2: Analisis de actividad sismica en Perú

Este informe presenta un análisis de los sismos ocurridos en Perú, centrado en su localización, magnitud, profundidad y la red de estaciones sísmicas utilizadas para su monitoreo. Perú, ubicado en una zona tectónicamente activa debido a la subducción de la placa de Nazca bajo la placa Sudamericana, experimenta frecuentes movimientos sísmicos.

Herramientas Utilizadas:

Este análisis y las visualizaciones fueron creados utilizando Tableau y Microsoft Excel, lo que permite integrar datos en tiempo real y realizar análisis detallados
